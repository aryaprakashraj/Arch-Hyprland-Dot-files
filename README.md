# Arch-Hyprland-Dot-files
An Minimal yet functional and efficient Arch Hyprland setup. Really Minimal, that you don't realize it has configs installed.

# Features

  Hyprland with active/inactive window opacity control  
  Minimal borders and 8px rounded corners
  Dwindle tiling layout
  Smooth animations with balanced speed
  Fuzzel launcher bound to SUPER + SPACE
  Waybar with a simple, clean theme
  Hyprpaper for wallpaper management
  Hypridle for screen idle handling
  Consistent spacing and sizing based on the 8-point grid approach

# Requirements

These packages must be installed for the setup to function:

Core:
  hyprland
  hyprpaper
  hypridle
  hyprlock (optional)
  waybar
  fuzzel
  kitty (or your preferred terminal)
  xdg-desktop-portal-hyprland
  pipewire / wireplumber (audio)
  wl-clipboard

Optional:
  grim + slurp (screenshots)
  wlogout (logout menu)
  swww (alternative wallpaper daemon)
  tumbler (thumbnailer)
